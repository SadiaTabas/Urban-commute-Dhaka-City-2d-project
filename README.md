 
# 🌆 Scene-1: Urban Commute – Dhaka City (OpenGL Project)

An interactive **2D city simulation scene** built using **OpenGL and GLUT**, inspired by real-life urban life in **Dhaka City**. The project demonstrates computer graphics concepts such as transformations, animations, and event-driven interactions in a visually rich environment.

 

#  Introduction

This project presents a fully interactive urban city scene that combines multiple real-world elements such as roads, buildings, transportation systems, and natural environments into a single integrated OpenGL application.

The scene includes **dynamic animations, weather effects, and user-controlled interactions**, making it both educational and visually engaging.

It demonstrates strong understanding of:

* OpenGL primitives
* Transformations
* Timer-based animations
* Event-driven programming (keyboard & mouse input)

---

# 🏙️ Scene Description

The environment is inspired by **Dhaka City** and includes:

* Roads and bridges
* Educational institutions (school)
* Hospital complex
* Residential and commercial buildings (5 total)
* River with animated waves
* Trees (4 total) and grass (3 areas)
* Electric wires and traffic lights
* Vehicles (cars, bus, train, ship)

# 🎬 Animations

The project includes multiple real-time animations:

* 🚗 Moving vehicles (cars, bus, train, ship)
* ☁️ Cloud movement (left → right & right → left)
* 🌊 Animated river waves
* 🌞🌙 Day–Night cycle (sun and moon transition)
* 🌇 Sunset animation
* 🌧️ Rain animation
* ❄️ Snowfall animation

All animations are implemented using **timer-based functions** for smooth continuous motion.

---

# 🖱️ Mouse Controls

* **Left Click** → Toggle rain animation 🌧️
* **Right Click** → Toggle snow animation ❄️

---

# ⌨️ Keyboard Controls

| Key     | Action                       |
| ------- | ---------------------------- |
| `N / n` | Toggle Day / Night mode 🌞🌙 |
| `R / r` | Start sunset animation 🌇    |
| `M / m` | Start train movement 🚆      |
| `S / s` | Stop train movement 🛑       |
| `L / l` | Move clouds left → right ☁️  |
| `J / j` | Move clouds right → left ☁️  |
| `K / k` | Stop cloud movement ⛅        |
